# Diplom Desk - Django Project

**Diplom Desk** — это платформа для размещения объявлений и отзывов, с возможностью аутентификации пользователей через JWT и функционалом восстановления пароля по электронной почте.

## Описание проекта

Проект предоставляет пользователям возможность:
- Размещать объявления о продаже товаров.
- Оставлять отзывы о товарах.
- Проходить процесс регистрации и аутентификации через email.
- Восстанавливать пароль через email.

Платформа также использует различные инструменты, такие как Django, PostgreSQL, JWT для аутентификации и CORS для разрешения кросс-доменных запросов.

## Технологии
- **Django** — веб-фреймворк для разработки серверной части.
- **PostgreSQL** — база данных для хранения информации.
- **Django Rest Framework (DRF)** — для создания API.
- **simplejwt** — для аутентификации через JSON Web Tokens.
- **CORS Headers** — для разрешения кросс-доменных запросов.
- **docker** — для контейнеризации приложения.

## Установка и настройка
### Клонирование репозитория
```bash
git clone https://github.com/RafaelManasyan/Diplom_Desk.git
```

### Создание и активация виртуального окружения
Для создания виртуального окружения и активации его, выполните следующие команды:
```
python3 -m venv .venv
source .venv/bin/activate  # Для macOS/Linux
.venv\Scripts\activate  # Для Windows
```
### Установка зависимостей
Установите все необходимые зависимости с помощью pip:
```
pip install -r requirements.txt
```
### Настройка базы данных

Вам нужно настроить подключение к базе данных PostgreSQL в файле settings.py:
```DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.postgresql',
        'NAME': 'your_db_name',
        'USER': 'your_db_user',
        'PASSWORD': 'your_db_password',
        'HOST': 'localhost',
        'PORT': '5432',
    }
}
```
### Миграции
Примените миграции для создания таблиц в базе данных:

```python manage.py migrate```

### Запуск проекта

Для запуска проекта локально используйте следующую команду:
```python manage.py runserver```

## Docker

Для работы с проектом в Docker, выполните следующие шаги:
	1.	Создайте образ и запустите контейнеры:
```docker-compose up --build```
	2.	Откройте браузер и перейдите по адресу http://localhost:8000/.

##  Swagger для API

Документация API доступна через Swagger по адресу:
http://localhost:8000/swagger/
